Sobre o Projeto

Objetivo

Este projeto foi desenvolvido com o objetivo de apresentar informações sobre a mitologia grega de forma organizada, visual e interativa.

Tema

O site aborda deuses, heróis, criaturas mitológicas e histórias da Grécia Antiga.

Tecnologias Utilizadas

O site foi desenvolvido utilizando HTML, CSS e JavaScript para criar uma experiência semelhante ao da wikipedia, com uma estrutura de navegação clara e um design na medida do possivel.

Motivação do desenvolvimento

Projeto criado para a disciplina do fessô Coelho.
